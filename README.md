# AI Web KPIs

Simple static web app for admin member management and KPI dashboards.

## الملفات
- `admin.html`: صفحة الأدمن لحفظ واستيراد بيانات الأعضاء وتعديلها.
- `index.html`: صفحة الموقع للمستخدمين لعرض بياناتهم ومؤشرات الأداء.
- `app.js`: منطق تسجيل الدخول، تخزين البيانات في `localStorage`، واستيراد Excel.
- `styles.css`: تنسيق واجهة المستخدم.

## كيفية الاستخدام
1. افتح `admin.html` في المتصفح.
2. سجّل دخول الأدمن باستخدام:
   - اسم المستخدم: `admin`
   - كلمة المرور: `ebdaa`
3. استخدم أزرار استيراد Excel لإضافة أو تحديث بيانات الأعضاء.
4. افتح `index.html` لتسجيل دخول الأعضاء وقراءة بياناتهم ومؤشرات الأداء.

## ملاحظات
- يتم حفظ جميع البيانات محلياً في `localStorage`.
- تسجيل دخول العضو يبقى مفتوحاً حتى يضغط المستخدم على زر "تسجيل الخروج".
- البيانات ترتبط مباشرة بين صفحة الأدمن وواجهة العضو عند تعديلها.

## تنسيق ملف Excel المطلوب
يجب أن تحتوي الأعمدة على العناوين التالية:
- `اسم العضو`
- `الرقم السري`
- `الرقم`
- `الرقم القومي`
- `المحافظة`
- `رقم الغرفة`
- `حضور التدريب الاول`
- `استلام المشروع الاول`
- `حضور التدريب الثاني`
- `حضور التدريب الثالث`
- `المشروع الثاني`
- `المشروع الثالث`
- `استلام المشروع النهائي`

يمكنك فتح المشروع باستخدام أي خادم محلي أو بإطلاق `Live Server` في VS Code.
