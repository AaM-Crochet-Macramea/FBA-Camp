# منصة الأعضاء والتدريبات

## المرئية العامة
منصة إدارة الأعضاء ومؤشرات الأداء مع نظام تتبع التدريبات والمشاريع.

## المستخدمون
- **الأعضاء**: عرض بيانات التدريبات والمشاريع الخاصة بهم
- **الإدارة**: إدارة الأعضاء ومؤشرات الأداء وتحميل البيانات

## الهوية البصرية
- **الطراز**: عصري وحديث (Modern Dark Theme)
- **الوضع**: الوضع الليلي بشكل أساسي (Dark Mode)
- **المشاعر**: احترافي، موثوق، سهل الاستخدام
- **المبادئ**: بساطة، وضوح، كفاءة

## المراجع المرغوبة والمرفوضة
- **مستوحى من**: Vercel, GitHub Dark, Figma Dark
- **تجنب**: تصاميم قديمة، تشويش بصري، ألوان متضاربة

## اللون والنبرة
- **النبرة**: مهنية، ودية، واضحة
- **الجمهور**: متخصصون تقنيون وإداريون
